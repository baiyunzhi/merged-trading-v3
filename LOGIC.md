# 合并版本 v3 · 完整交易逻辑

一笔交易从"选品"到"平仓结算"的完整决策链。参数见 `config/settings.json`。

## 决策链总览

```
多因子选品排序 → 市场环境过滤(ADX+布林带) → 结构信号(道氏123/双底双顶)
  → 量能+实体催化剂确认 → 仓位定额(ATR) → 板块限制检查 → 下一根开盘进场
  → 结构止损 → 保本 → 分批止盈 → 追踪止损 → 时间止损 → 平仓(分级滑点)
```

---

## 〇、多因子选品（搭建系，claude系所缺的能力）

先对全部品种打分排序，综合分 = 趋势40% + 动量35% + 波动25%：

- **趋势分**：MA5>MA10>MA20>MA60 多头排列 + ADX 强度（最多+40）
- **动量分**：近5日/20日涨跌幅 + RSI 位置（40-65加分，极端减分）
- **波动分**：ATR/价格在 0.8%-2.5% 最佳，过低过高都减分；量能放大额外加分

选出评分高的品种重点关注。

---

## 一、市场环境过滤（先决条件）

| 过滤器 | 阈值 | 含义 |
|--------|------|------|
| ADX | > 22 | 趋势强度，低于则为震荡市，不交易 |
| 布林带宽度 | > 0.05 | 波动率，过窄则行情死水，不交易 |

两个都通过才往下看信号。

## 二、入场信号（4选1 + 催化剂确认）

**催化剂**：成交量 ≥ 20日均量×1.5 且 实体 ≥ 0.6×ATR。

| 信号 | 方向 | 条件 |
|------|------|------|
| 123多头突破 | 多 | 道氏BULL状态 + HH+HL + 突破最近确认高点 |
| 123空头跌破 | 空 | 道氏BEAR状态 + LH+LL + 跌破最近确认低点 |
| 双底/三底突破 | 多 | 形态成立 + 突破颈线 |
| 双顶/三顶跌破 | 空 | 形态成立 + 跌破颈线 |

**执行**：信号收盘确认，**下一根K线开盘成交**（无前瞻偏差，区别于旧搭建系）。

## 三、仓位（取三者最小值）

1. 风险定额：每笔最多亏 2500 或账户 0.5%，按 ATR×2.5 止损距离反推
2. 保证金约束：可用保证金 ÷ 每手保证金
3. 组合风险预算：所有持仓总风险 ≤ 账户 2%

## 四、止损（结构止损）

取结构止损与 ATR 止损中**更紧**者：
- 结构止损：摆动点外侧 0.3×ATR
- ATR 止损：进场价 ± 2.5×ATR

## 五、出场（多重并行）

| 机制 | 规则 |
|------|------|
| 保本 | 浮盈 1.0R，止损移到成本 |
| 分批止盈 | 浮盈 2.0R，平 50% |
| 追踪止损 | 浮盈 2.5R 后，止损跟随收盘价 2.5×ATR |
| 时间止损 | 持仓 30 根K线浮盈<0.3R，平仓 |
| 道氏反转 | 跌破结构位/道氏转空，下根开盘平 |

## 六、组合风控（claude系，搭建系所缺）

- 同板块同向 ≤ 2 仓
- 单板块总风险 ≤ 账户 3%

## 七、成本（分级滑点）

| 场景 | 滑点 |
|------|------|
| 开仓 | 0.5 tick |
| 止损出场 | 2 tick（惩罚） |
| 手续费 | 3 元/手 |
